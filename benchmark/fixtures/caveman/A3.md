Task creation flows four layers:
1. API: POST /tasks -> create_task handler in taskflow/api/routes_tasks.py calls app.services.tasks.create_task(...).
2. Service: TaskService.create_task (taskflow/services/task_service.py) builds Task, calls self.tasks.add(task), then notifies assignee.
3. Repository: BaseRepository.add (taskflow/repositories/base_repo.py) turns entity into row, calls self.db.insert(self.table, row).
4. DB: Database.insert (taskflow/db.py) assigns id, stores row. New id returned back up chain.
