Project layered Python package. `taskflow/` holds: top-level `config.py`, `db.py`, `cli.py`; `models/` (base, user, task, project); `repositories/` (base_repo, user_repo, task_repo, project_repo); `services/` (auth, task, project, notification); `api/` (app, deps, routes_users, routes_tasks, routes_projects); `utils/` (validation, dates, logging). `tests/` holds four test modules: repositories, auth, tasks, API.
